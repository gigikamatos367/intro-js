# Introdução ao JavaScript

**JavaScript** ou **JS** é uma linguagem de programação usada para criar páginas web dinâmicas, interativas e inteligêntes. 

Também é possível criar aplicações no servidor usando **Node.js**.

---

## Variáveis

```javascript
let nome = "Ana";
const idade = 20;
var antigo = true; 
```
